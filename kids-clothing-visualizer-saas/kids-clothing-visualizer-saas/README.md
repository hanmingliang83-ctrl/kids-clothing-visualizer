# 🧒 童装搭配可视化智能体

> **Kids Clothing Visualizer** — AI驱动的童装搭配专家，输入孩子信息+场景，自动生成搭配方案+效果图

## 📋 项目概述

本智能体将【童装搭配知识库】与【AI文生图】合二为一，实现 **"输入孩子信息→搭配推理→生成效果图"** 的全自动流程。

### 核心能力

| 功能 | 说明 |
|------|------|
| 🧠 **智能搭配** | 基于200万字童装知识库的底层规则引擎 |
| 🎨 **AI生图** | 集成阿里云通义万相2.7，文本描述→搭配效果图 |
| 👶 **体型适配** | 支持偏胖/偏瘦/标准体型，自动调整版型建议 |
| 🌡️ **26℃法则** | 温度×场合×活动量，三层洋葱式精准穿搭 |
| 🛡️ **安全红线** | GB31701国标绳带、配件、面料等级自动检测 |
| 📱 **所见即所得** | 文字方案 + 效果图同步输出 |

### 适用场景

- 家长日常搭配（上学/运动/旅行/派对/正式场合）
- 幼儿园/学校活动穿搭建议
- 换季衣橱规划
- 特殊体型/过敏体质儿童穿搭

## 🚀 快速开始

### 1. 环境要求

- Python 3.12+
- 阿里云百炼API Key

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置API密钥

```bash
export BAILIAN_API_KEY="sk-你的API密钥"
```

### 4. 运行示例

```bash
python examples/demo_spring_outing.py
```

### 5. 运行API服务

```bash
cd deploy
docker-compose up -d
```

## ☁️ SAAS部署

### Docker部署（推荐）

```bash
cd deploy
docker-compose up -d
```

服务将运行在 `http://localhost:58080`

### API接口

**POST /api/recommend**
```json
{
  "age": 7,
  "gender": "男童",
  "weight_kg": 27.5,
  "height_cm": 130,
  "body_type": "偏胖",
  "season": "夏季",
  "occasion": "春游",
  "temperature_c": "12-22",
  "generate_image": true
}
```

**响应：**
```json
{
  "text_plan": "🎯 春日游学·活力运动风...",
  "image_url": "https://...",
  "rules_applied": ["规则8-7", "规则9-2", "规则10-4"]
}
```
